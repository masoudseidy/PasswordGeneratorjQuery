أولا برای استفاده از این افزونه ابتدا باید JQuery را به پروژه خود اضافه کنید چرا که این افزونه با JQuery نوشته شده است و سپس افزونه را به پروژه خود اضافه کنید. 

<scrtip src="../jquery.js"></script>

ثانياَ یک فیلد متن به همراه یک دکمه برای تغییر رمز عبور در درون یک تگ div قرار دهید و به این تگ div و به این تگ div یک مقدار به Class یا id بدهید 

<div id="input-example">
  <input type="text" class="input-generator-pass">
  <button>تغيير</button>
</div>

ثالثا حالا با استفاده از این مقداری که به مشخصه id تگ div داده اید تابع generatePassword() را فراخوانی کنید. 

$('.input-example').generatePassword();

رابعا برخی از option های تابع generatePassword();
به عنوان مثال برای تعیین طول رمز عبور و یا برای انتخاب حروف، برچسب ها و اعداد که می خواهید در رمز عبور باشد

passCharachterSet
این پارامتر برای تعیین نوع کاراکترهای خاص استفاده می شود، مثلا اینکه آیا آنها حروف یا اعداد هستند. این ویژگی مقادیر 0-9 را قبول می کند و a-z ، A-Z و همچنین کاراکترهای خاص را می پذیرد.

passSize
این پارامتر طول پسورد را مشخص می کند.

passSeparate
در صورتی که بخواهید از چند گروه کارکتری در داخل پسورد استفاده کنید باید آنها را با کارکتر خاصی از هم جدا کنید و به تابع generatePassword() هم این کارکتر را بشناسانید برای این کار باید مقدار پارامتر passSeparate را برابر با یک کارکتر دلخواه کنید میتوان از کاراکتر _ یا space و یا هر کاراکتر دلخواه دیگری مثل @ استفاده کنید
به عنوان مثال اگر شما میخواهید در پسوردتان از کاراکتر های a-z , A-Z استفاده کنید باید به صورت زیر بنویسید

a-z SeparateValue A-Z

که در این صورت به جای عبارت SeparateValue مقداری که برای پارامتر passSeparate قرار داده بودید را وارد می کنید. 
passSeparate: '_',
passCharachterSet:'a-z_A-z'


$('div#input-example').generatePassword({
    passSeparate: ' '
    passCharachterSet: 'a-z #', // استفاده از کارکترهای a تا z و کاراکتر #
    passSize: 18
});
